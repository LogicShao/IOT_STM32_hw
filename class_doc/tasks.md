# 基于 STM32F070CBT6TR+HT7017 的直流电能表开发板实验

## 一、实验目的

1. 掌握基于 STM32CubeMX+CLion 的嵌入式开发流程，熟悉 STM32F070CBT6TR 开发板的使用。
2. 理解并掌握 GPIO、定时器、外部中断等外设的配置与使用方法，掌握有限状态机（FSM）在嵌入式系统中的应用。
3. 学习 GN1650 数码管驱动芯片与 HT7017 直流计量芯片的工作原理，掌握 UART、I2C 通信协议及外设 API 的调用方法。
4. 了解 RS485 半双工通信原理及 HC-05 蓝牙模块的使用方法，实现开发板与上位机或手机端的数据交换。
5. 培养综合调试能力，包括硬件连接、代码烧录、串口通信调试及系统联调。

## 二、实验设备与环境

硬件：基于 STM32F070CBT6TR+HT7017 的直流电能表开发板、SEGGER J-Link 调试器、RS485 转 TTL 模块、CH340 串口模块、HC-05 蓝牙模块、面包板、杜邦线等。

软件：Windows 10/11、CLion、STM32CubeMX、STM32CubeCLT、OpenOCD 与 J-Link 驱动、串口调试助手 XCOM、HC 蓝牙助手 APP 等。
